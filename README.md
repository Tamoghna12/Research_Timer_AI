### Enable AI Features (Optional)
Unlock powerful AI-generated session summaries by connecting your preferred AI service:

#### Option 1: OpenAI (GPT-4, GPT-3.5) - Recommended
1. **Get API Key**: Visit [OpenAI API Keys](https://platform.openai.com/api-keys)
2. **Create Account**: Sign up if you don't have an OpenAI account
3. **Generate Key**: Click "Create new secret key" and copy it
4. **Add to App**: Go to Settings ‚Üí AI Features ‚Üí paste your API key
5. **Test Connection**: Click "Test Connection" to verify setup

**Cost**: ~$0.01-0.05 per session summary (very affordable)

#### Option 2: Anthropic (Claude) - Great Alternative
1. **Get API Key**: Visit [Anthropic Console](https://console.anthropic.com/)
2. **Create Account**: Sign up for Anthropic API access
3. **Generate Key**: Create new API key in console
4. **Configure**: Settings ‚Üí AI Features ‚Üí select Anthropic ‚Üí add key

#### Option 3: Google Gemini - Free Tier Available
1. **Get API Key**: Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
2. **Create Key**: Sign in with Google account and generate API key
3. **Setup**: Settings ‚Üí AI Features ‚Üí select Google Gemini ‚Üí add key

#### Option 4: Groq - Ultra-Fast Inference
1. **Get API Key**: Visit [Groq Console](https://console.groq.com/keys)
2. **Create Account**: Sign up for Groq API access
3. **Generate Key**: Create new API key
4. **Configure**: Settings ‚Üí AI Features ‚Üí select Groq ‚Üí add key
